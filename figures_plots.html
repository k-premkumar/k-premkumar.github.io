Figures/Plots

Figures

Figures play an important role in illustrating and understanding concepts/systems. Hence, one should use a good software for drawing figures. I recommend inkscape, which is used to produce scalable vector graphics pictures.

Plots

Engineering results need to be plotted for performance analysis and design. I recommend PGFPlots (or PGF/TikZ) and gnuplot (see, e.g., gnuplot_script_example.plt), if you are a open source enthusiast, or Matlab.

Note:

  • Generate figures and plots in all the following formats: .eps, .pdf, and .jpg as you will need .eps and .pdf for LaTeX documents and .jpg for PowerPoint presentations.

  • To convert images from one format to another, there exist softwares like GIMP, inkscape.
